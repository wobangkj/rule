#### 公共约定

- 状态码  

1.200(创建/修改/删除)  
>历史遗留规范: 创建: 201, 修改: 206, 删除: 209  
```js
{
    "status": 200,
    "msg": "请求成功"
}
```

```js

    "msg":"请求成功",

         "other":"other"
        }

        "client_page":1,
```

